# ⚡ Démarrage Ultra-Rapide

## 🎯 3 Étapes pour Commencer

### 1. Installer les dépendances
```bash
npm install
```

### 2. Lancer l'application
```bash
npm start
```
→ Ouvre http://localhost:4200

### 3. Lire la documentation
- **[COMMENCER_ICI.md](COMMENCER_ICI.md)** - Instructions complètes
- **[PROJET_ANGULAR.md](PROJET_ANGULAR.md)** - Guide détaillé

---

## ⚙️ Configuration Firebase (À faire avant le déploiement)

1. Mettre à jour `src/environments/environment.ts` avec vos clés
2. Mettre à jour `.firebaserc` avec votre project ID
3. Créer les collections `contacts` et `quotes` dans Firestore

---

## 🚀 Déploiement

```bash
npm run build:prod
firebase deploy --only hosting
```
