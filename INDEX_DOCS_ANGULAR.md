## 🗺️ Parcours Recommandé

### 1. Premier Lancement (30 minutes)
1. Lire [COMMENCER_ICI.md](COMMENCER_ICI.md)
2. Exécuter `npm install`
3. Exécuter `npm start`
4. Explorer l'application sur http://localhost:4200

### 2. Comprendre le Projet (1 heure)
1. Lire [PROJET_ANGULAR.md](PROJET_ANGULAR.md)
2. Explorer [STRUCTURE_PROJET.md](STRUCTURE_PROJET.md)
3. Parcourir le code dans `src/app/`

### 3. Configuration Firebase (30 minutes)
1. Section Firebase dans [PROJET_ANGULAR.md](PROJET_ANGULAR.md)
2. Mettre à jour `src/environments/environment.ts`
3. Mettre à jour `.firebaserc`
4. Créer les collections Firestore

### 4. Personnalisation (selon besoins)
1. Modifier le contenu dans `src/app/pages/`
2. Ajuster les styles dans `src/styles.css`
3. Voir [PROJET_ANGULAR.md](PROJET_ANGULAR.md) - Section "Personnalisation"

### 5. Déploiement (1 heure)
1. Lire [GUIDE_DEPLOIEMENT_ANGULAR.md](GUIDE_DEPLOIEMENT_ANGULAR.md)
2. Build: `npm run build:prod`
3. Tester localement
4. Déployer: `firebase deploy`
